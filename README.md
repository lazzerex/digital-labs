## Development

### Prerequisites

- Modern web browser
- Local web server (optional)

### Setup

1. Clone the repository:
```bash
git clone https://github.com/lazzerex/digital-labs.git
```

2. Open `index.html` in your browser or serve through a local web server

- HTML5
- CSS3 with Custom Properties
